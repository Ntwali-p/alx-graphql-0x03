# ALX GraphQL 0x03 - Rick and Morty App

This project is part of the ALX GraphQL module. It demonstrates the usage of GraphQL queries in a React/Next.js application to fetch data from the Rick and Morty API.

## Features

- Query characters, episodes, and locations using GraphQL
- Display results in a responsive UI
- Implemented an **ErrorBoundary** component to catch JavaScript errors
